About CyberGuard

CyberGuard is a Password Strength Analyzer designed to help users create secure and reliable passwords.

The system evaluates password strength based on important security factors such as password length, uppercase letters, numbers, and special characters.

The main objective of this project is to improve cybersecurity awareness and encourage users to use stronger passwords to protect their online accounts from cyber threats.

This project was developed using HTML, CSS, and JavaScript as part of a Cyber Security Internship project.
